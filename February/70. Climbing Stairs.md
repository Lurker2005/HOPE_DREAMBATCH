## Dynamic Programming

```java
class Solution {
    public int climbStairs(int n) {
        if (n <= 3) return n;

        int prev1 = 3;
        int prev2 = 2;
        int cur = 0;

        for (int i = 3; i < n; i++) {
            cur = prev1 + prev2;
            prev2 = prev1;
            prev1 = cur;
        }

        return cur;        
    }
}
```

## Math Based

```java
class Solution {
    public int climbStairs(int n) {
        double sqrt5 = Math.sqrt(5);
        double l = (1 + sqrt5) / 2;
        double sl = (1 - sqrt5) / 2;

        return (int)Math.round(
            (Math.pow(l, n + 1) - Math.pow(sl, n + 1)) / sqrt5
        );
    }
}
```
