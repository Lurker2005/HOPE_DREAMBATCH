## Array
```java
class Solution {
    public int[] searchRange(int[] nums, int target) {
        int count=0;
        int ind=0;
        for(int i=0;i<nums.length;i++){
            if(nums[i]==target){
                ind=i;
                count++;
            }
            if(nums[i]>target){
                break;
            }
        }
        if(count==0){
            return new int[]{-1,-1};
        }
        return new int[]{ind-count+1,ind};
    }
}
```
