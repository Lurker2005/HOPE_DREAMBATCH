## Binary Search
```java
class Solution {
    public boolean isPerfectSquare(int num) {
        int left=2;
        int right=(int)num/2;
        if(num==1){
            return true;
        }
        while(left<=right){
            int mid=(left+right)/2;
            if(mid*mid==num){
                return true;
            }
            else if(mid < num / mid){

                left=mid+1;
            }
            else{
                right=mid-1;
            }
        }
        return false;
    }
}
```

class Solution {
    public boolean isPerfectSquare(int num) {
        if(num==1){
            return true;
        }
        else{
            int val=1;
            while(num>0){
                num=num-val;
                val+=2;
            }
            if(num==0){
                return true;
            }

        }
        return false;
        
    }
