## aTwo pointer

```java
class Solution {
    public void sortColors(int[] nums) {
        int i = 0, mid = 0,j = nums.length-1;

        while(mid <= j ){
            if(nums[mid] ==0){
                swap(nums,mid,i);
                i++;
                mid++;
            }
            else if(nums[mid] == 1){
                mid++;
            }
            else{
                swap(nums,mid,j);
                j--;
            }
        }

    }

    void swap(int[] arr,int i, int j){
        int temp = arr[i];
        arr[i] = arr[j];
        arr[j] = temp; 
    }
}
```

## Array

```java
class Solution {
    public int[] sortColors(int[] nums) {
        Arrays.sort(nums);
        return nums;
        
    }
}
```
