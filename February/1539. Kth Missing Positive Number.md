## Math way

```java
class Solution {
    public int findKthPositive(int[] arr, int k) {
        for (int i = 0; i < arr.length; i++) {
            int mis = arr[i] - (i + 1);

            if (mis >= k) {
                return k + i;
            }
        }

        int totalMissing = arr[arr.length - 1] - arr.length;
        return arr[arr.length - 1] + (k - totalMissing);
    }
}

```

## Array while loop

```java
class Solution {
    public int findKthPositive(int[] arr, int k) {
        int missing = 0;
        int i = 1;
        int j = 0;

        while (true) {
            if (j < arr.length && arr[j] == i) {
                j++; 
            } else {
                missing++; 
                if (missing == k) {
                    return i;
                }
            }
            i++;
        }
    }
}
```
